# "@小样燃剪"水印去除优化报告

## 执行摘要

经过 14 次迭代优化，当前系统可以检测并去除"@小样燃剪"水印，但存在以下限制：

### 当前能力
- **水印检测**：使用 CLAHE + HSV 融合检测算法，可以检测右上角水印和底部字幕
- **图像修复**：支持多种 OpenCV inpainting 算法（Telea, NS, Multi-pass, Aggressive）
- **PyTorch 支持**：已安装 PyTorch，具备集成深度学习模型的基础

### 技术限制
1. **检测精度**：水印与头发/背景对比度低，难以完全区分
2. **修复质量**：OpenCV 传统算法无法完美重建复杂背景纹理
3. **当前效果**：水印变淡但仍可见，无法达到"完美去除"

---

## 迭代历史

### 迭代 1-3：基础检测
- 方法：多阈值检测 + 形态学操作
- 结果：检测到 2-11% 区域，水印残留严重
- 问题：检测不完整，mask 覆盖不足

### 迭代 4-6：ROI 优化
- 方法：聚焦右上角 ROI，使用 Otsu 自适应阈值
- 结果：检测到 3-5% 区域，底部字幕检测良好
- 问题：右上角文字检测仍不完整

### 迭代 7-9：激进策略
- 方法：降低阈值、扩大检测区域、边界框扩展
- 结果：检测到 12-27% 区域
- 问题：覆盖太多非水印区域（头发），修复后出现模糊

### 迭代 10-12：智能过滤
- 方法：边缘密度分析、纹理过滤
- 结果：过度过滤导致漏检
- 问题：水印与头发特征相似，无法可靠区分

### 迭代 13-14：融合检测
- 方法：CLAHE + HSV 双路径检测 + 形态学融合
- 结果：检测到 6-9% 区域，最佳平衡
- 问题：仍无法完整覆盖水印，修复质量受限

## 根本挑战

### 1. 图像特征分析
| 特征 | 水印 | 头发 | 可区分度 |
|------|------|------|----------|
| 亮度 | 高 (150-225) | 高 (100-200) | 低 |
| 饱和度 | 低 (<50) | 中 (50-100) | 中 |
| 边缘密度 | 中 | 高 | 中 |
| 纹理 | 简单 | 复杂 | 低 |
| 位置 | 右上角 | 全图 | 高 |

### 2. OpenCV Inpainting 限制
- **Telea**: 基于快速行进算法，适合小孔洞填充
- **Navier-Stokes**: 基于流体力学，适合平滑区域
- **限制**: 无法重建复杂纹理（如头发细节）

## 建议方案

### 方案 A：集成深度学习 Inpainting 模型（推荐）
**优点**：可以学习并重建背景纹理
**模型选择**：
1. **LaMa** (Large Mask Inpainting) - 效果最佳
2. **AOT-GAN** - 适合不规则 mask
3. **MAT** (Mask-Aware Transformer) - 最新 SOTA

**实现步骤**：
```bash
pip install lama-cleaner
# 或下载预训练模型
wget https://github.com/Sanster/models/releases/download/add_big_lama/big-lama.pt
```

### 方案 B：视频多帧融合（针对视频）
**原理**：利用时间冗余，从无水印帧借用信息
**实现**：
1. 检测视频中无水印的关键帧
2. 光流法对齐帧
3. 多帧融合修复

### 方案 C：用户辅助标注
**原理**：让用户提供水印模板或手动标注
**实现**：
1. 提供 GUI 工具手动标注水印区域
2. 使用标注结果生成精确 mask
